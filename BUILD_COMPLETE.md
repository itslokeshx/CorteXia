## How to Preview & Deploy

### Preview
1. Click the **Version Box** in the top-right to see the live preview
2. Navigate through all pages using the sidebar
3. Create new entries using the "New" buttons
4. Try different filters and views
5. Toggle between light/dark modes

### Deploy to Vercel
1. Click the **Publish** button in the top-right
2. Connect to your Vercel account
3. Your app will be live on a Vercel URL

### Download Code
1. Click the **three dots** in the top-right
2. Select "Download ZIP"
3. Extract and run:
   ```bash
   npm install
   npm run dev
   ```
